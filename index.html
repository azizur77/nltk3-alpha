NLTK 3.0 Alpha Releases

NLTK 3.0 is the first version of NLTK to support Python 3. This page contains links to the most recent alpha release.

Python 3 support is a work in progress, led by Mikhail Korobov, and with support from the Python Software Foundation. The alpha releases are only intended for people who want to support the development effort by testing releases and submitting bug reports and patches. We will acknowledge such support in our final release announcement.

The NLTK 3.0.X series will support Python 2.6 and 2.7 as well as Python 3. The online book will be updated in due course.

The most significant API changes are as follows: corpus readers now return Unicode strings by default; [MORE]. The following are dropped: Tkinter graphical interfaces; babelize; sourcedstrings; [MORE]
